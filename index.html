What is HTML?

  • Hyper Text Markup Language
  • NOT a programming language (does not have logic like JavaScript)
  • Used to create web pages and documents

HTML Tags

  • Element names are surrounded by angle brackets
  • Tag elements usually come in pairs (start and end tag)
  • Some tags close themselves

head Tag

Used to store the information about a webpage such as metadata, title of the page and links to CSS and JavaScript files.
